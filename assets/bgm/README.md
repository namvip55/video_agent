# BGM (Background Music) Assets

Place `.mp3` files here for manga video background music.

## Expected files:
- `default.mp3` — Default BGM for any genre
- `comedy.mp3`  — Hài hước
- `action.mp3`  — Hành động
- `romance.mp3` — Tình cảm
- `mystery.mp3` — Trinh thám
- `chill.mp3`   — Nhẹ nhàng
- `epic.mp3`    — Hoành tráng
- `lofi.mp3`    — Lo-fi beats

## Notes:
- Files should be royalty-free or licensed for use
- Recommended: 60-120s length (will be looped if shorter than video)
- Format: MP3, 128-320kbps
- Pipeline applies fade in (2s) and fade out (3s) automatically
